# Installing Agon

This page explains how to install Agon on your computer so you can start managing your studio.

## What you need

- A Windows, macOS, or Linux computer that stays on during studio hours
- An internet connection
- About 10 minutes for the initial setup

## Steps

1. Download the Agon installer from the [Agon GitHub releases page](https://github.com/agon-studio/agon/releases).
   Choose the version for your operating system (Windows, macOS, or Linux).

2. Run the installer and follow the on-screen instructions.

3. When the installation is complete, open Agon from your applications menu or desktop shortcut.

4. Agon will start the setup wizard automatically. Follow the [Onboarding Guide](onboarding) to complete setup.

## What if something goes wrong?

**The installer does not start**
Make sure you downloaded the correct version for your operating system. On macOS, you may need to approve the app in System Settings > Privacy & Security.

**The app opens but shows an error**
Check that no other application is using port 8000 on your computer. Agon's internal server uses this port.
