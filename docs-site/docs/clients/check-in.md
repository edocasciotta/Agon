# Checking in to a class

When you arrive at the studio for a class, you need to check in. This confirms your attendance and lets the studio know you're there. This guide explains how.

---

## Before you start

You need a **confirmed booking** for the class you're attending. If you haven't booked yet, see [Booking a Class](booking-a-class).

---

## Check-in window

Check-in is only available for a set time window around the class start. Your studio sets this window — the typical defaults are:

- Check-in **opens** 30 minutes before the class starts
- Check-in **closes** 15 minutes after the class starts

For example: for a class at 10:00 AM, you can check in from 9:30 AM to 10:15 AM.

If you try to check in outside this window, you'll see a message telling you when the window opens or that it has closed. Ask the reception staff to check you in manually if needed.

---

## Method 1 — App check-in

This is the simplest method: you check yourself in directly from the app.

1. Open the **Agon Studio** app.
2. Go to **My Bookings**.
3. Find today's class — it will show a **Check In** button when the check-in window is open.
4. Tap **Check In**.
5. You'll see a confirmation screen: *"You're checked in to [class name]!"*

That's it — no QR code, no staff involvement needed.

---

## Method 2 — QR code check-in

If your studio uses a QR scanner at reception (or a self-service kiosk), you can check in by showing your QR code.

1. Open the **Agon Studio** app.
2. Go to **My Bookings**.
3. Tap the class you're attending.
4. Tap **Show QR code**.
5. Hold your phone screen in front of the scanner at reception.

The scanner reads your code and confirms your check-in automatically.

---

## What if something goes wrong?

**The Check In button isn't showing**
The check-in window may not be open yet — it opens a set number of minutes before the class starts. Check the class time and try again closer to the start. If the class has already started and you're past the closing time, ask the receptionist or instructor to check you in manually.

**The app says "No booking found"**
This means the app can't find a confirmed booking for this class. Make sure you have a booking for the correct class on the correct date. If you're sure you booked, check that your phone has an internet connection and try again.

**The QR code scanner at reception is not working**
Ask the staff member to check you in manually from their desktop app.
