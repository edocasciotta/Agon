# Managing classes

This page explains how to create class templates and schedule classes for your clients to book.

## Before you start

Agon uses a two-step system for classes:
1. First, create a **class template** (e.g. "Vinyasa Flow") with default settings
2. Then, **schedule** that template on specific dates and times

This saves time when you run the same type of class regularly.
