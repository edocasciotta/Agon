# Managing Classes

This page explains how to create class types, schedule classes on your calendar, manage the roster, and cancel classes when needed.

## How class scheduling works

Agon uses a two-step system:

1. **Class Types** — define a reusable class type (e.g. "Vinyasa Flow") with its default settings. In Italian: *Tipi di lezione*.
2. **Scheduled classes** — place that class type on your calendar on specific dates and times.

This means you set up each class type once, then schedule as many instances as you need without re-entering all the details each time.

---

## Creating a class type

Before you can schedule a class, you need to create a class type.

1. Go to **Class Types** in the left sidebar. In Italian: *Tipi di lezione*.
2. Click **+ New type**. In Italian: *+ Nuovo tipo*.
3. Fill in the details:
   - **Name** — the class name your clients will see (e.g. "Pilates Mat", "HIIT", "Vinyasa Flow")
   - **Description** — a short optional description
   - **Duration (min)** — how long the class runs, in minutes (e.g. 60)
   - **Default capacity** — maximum number of spots available (you can override this per session)
   - **Default instructor** — pre-selects an instructor when scheduling; can be changed per session
   - **Colour** — choose a colour for this class type on the calendar view
4. Click **Save class type**. In Italian: *Salva tipo di lezione*.

The class type is now available to use when scheduling classes.

### Editing a class type

Go to **Class Types**, find the type, and click **Edit** (pencil icon). Changes only affect future classes you schedule — they do not update existing scheduled classes.

### Deactivating a class type

If you no longer run a particular class type, click the deactivate option. It will no longer appear when scheduling new classes. Existing scheduled classes are not affected.

## Scheduling a single class

1. Go to the **Calendar** view. In Italian: *Calendario*.
2. Click the **Schedule class** button (top right), or click directly on a time slot in the calendar. In Italian: *Pianifica lezione*.
3. Select a **Class type** from the dropdown. The fields are pre-filled from the class type.
4. Set the details for this specific session:
   - **Date** — the date of the class
   - **Start time** — the exact start time. In Italian: *Ora di inizio*.
   - **Duration (min)** — duration in minutes
   - **Capacity** — override the default if needed
   - **Instructor** — change the instructor for this session
   - **Notes** — internal notes visible only to you and instructors
5. Click **Schedule class**. In Italian: *Pianifica lezione*.

The class immediately appears on the calendar and becomes visible and bookable to your clients in the mobile app.

---

## Scheduling a recurring class

If you run the same class regularly — for example, every Monday and Wednesday at 7:00 PM:

1. Click **Schedule class** on the calendar and select the **Recurring class** tab. In Italian: *Lezione ricorrente*.
2. Select a class type and set the time, duration, capacity, and instructor.
3. Choose the **days of the week** the class runs (e.g. Monday, Wednesday).
4. Set the **first date** and **end date** for the recurring series.
5. Click **Schedule classes**. In Italian: *Pianifica lezioni*.

Each session is created individually. You can later edit or cancel any single session without affecting the rest of the series.

---

## Editing a scheduled class

1. Click on the class in the **Calendar** view.
2. A detail panel opens on the right. You can see the start time, end time, capacity, status, and notes.
3. To cancel the class, click **Cancel class**. In Italian: *Cancella lezione*.

---

## Cancelling a class

1. Click on the class in the **Calendar** view.
2. In the detail panel, click **Cancel class**. In Italian: *Cancella lezione*.
3. Confirm the action.

**What happens immediately:**
- The class status changes to **Cancelled**
- Every client who had a booking for this class has their booking automatically cancelled
- Credits that were deducted for those bookings are refunded to each client's membership
- Every affected client receives a push notification
- The waitlist for the class is cleared

Cancelled classes remain visible on the calendar so you have a complete record.

## Frequently asked questions

**How do I create a new class?**
Go to **Class Types** → click **+ New type** → fill in the details → click **Save class type**. Then go to **Calendar** → click **Schedule class** → select the class type → click **Schedule class**.

**I scheduled a class at the wrong time.**
Click on the class in the calendar. The detail panel opens — note the current settings. Cancel the class and create a new one with the correct time.

**Clients are not seeing a new class in the app.**
Check that the class status is **Scheduled** (not Cancelled or Completed) and that the class date is in the future.

**How do I schedule the same class every week?**
Use the **Recurring class** tab in the Schedule class modal. Set the days of the week, first date, and end date.

**How do I see who booked a class?**
Click on the class in the **Calendar** view to open the detail panel. The panel shows the capacity and current booking count.
