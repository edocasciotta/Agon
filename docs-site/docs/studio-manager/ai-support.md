# AI Support Assistant

This guide explains how to use the built-in AI assistant to get instant answers about how Agon works — without leaving the app.

## Before you start

The AI assistant is available from the moment Agon is installed. You do not need to set anything up. It is accessible from every screen in the application.

## What the AI assistant does

The AI assistant answers your questions about Agon in plain language. You can ask it things like:

- "How do I cancel a class?"
- "What happens to a client's credits when I cancel a booking?"
- "How do I export a report?"
- "How do I set up a recurring membership?"

It knows everything covered in this documentation: classes, bookings, memberships, check-in, reports, payments, settings, and more. It responds in whatever language you write in.

## What the AI assistant does NOT do

By default, the assistant only answers questions — it never makes changes to your data.

Specifically, it will **not**:

- Create or modify client profiles
- Process payments or change membership plans
- Access your clients' personal information
- Delete any data

If you ask it to perform one of these actions, it will explain how to do it yourself instead.

**Exception:** studio managers can turn on [AI Action Mode](./ai-actions) to let the assistant create scheduled classes from a plain-language request. This is opt-in, only available to the studio manager account, and limited to creating classes — see the [AI Action Mode](./ai-actions) guide for details.

## How to use it

1. Click the round chat button (💬) in the bottom-right corner of the screen.
   [SCREENSHOT: The floating chat button in the bottom-right corner of the Agon desktop app]

2. A chat panel titled "Agon AI Support" opens on the right side of the screen.
   [SCREENSHOT: The AI Support chat panel open, showing the welcome message]

3. Type your question in the text box at the bottom of the panel. Write in plain language — no special commands or keywords are needed.
   [SCREENSHOT: The text input box with an example question typed]

4. Press **Send** or press **Enter** on your keyboard to submit your question.

5. The assistant will reply within a few seconds. If your question is unclear, it may ask a follow-up question to help you better.
   [SCREENSHOT: An example question and answer exchange in the chat panel]

6. To close the panel, click the chat button again or click anywhere outside the panel.

## Frequently asked questions

**What languages does it speak?**
The assistant responds in the same language you write in. If you write in Italian, it answers in Italian. If you write in Spanish, it answers in Spanish. It supports all major languages.

**Does it have memory between sessions?**
No. Each time you open the app and start a new conversation, the assistant starts fresh. It does not remember questions you asked in a previous session. If you need to continue a previous conversation, you will need to repeat the relevant context.

**What if it gives a wrong answer?**
The assistant draws its knowledge from this documentation. If it gives an answer that does not match what you see in the app, trust what you see in the app. You can also ask the question in a different way — sometimes rephrasing helps. If you believe the answer is consistently wrong, contact support so the documentation can be updated.

**Can it help me with problems that are not covered in the documentation?**
The assistant's knowledge is limited to the Agon documentation. For issues outside of that scope — such as network problems, operating system errors, or billing questions — contact your Agon support channel or open a ticket on the Agon GitHub repository.

## What if something goes wrong?

**The assistant does not respond after I press Send**
Check your internet connection. The AI assistant requires an active internet connection to work. If your connection is fine, try closing and reopening the chat panel.

**The chat panel does not open when I click the button**
Try restarting the Agon application. If the problem continues, check that the Agon backend service is running. On macOS, you can do this by quitting Agon completely and reopening it from your Applications folder.

**The assistant says it cannot find information on my question**
Try rephrasing your question using different words. For example, instead of "how do I set up a plan", try "how do I create a membership". The assistant uses the exact terms used in this documentation, so matching that language helps.
